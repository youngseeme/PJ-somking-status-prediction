# 생체지표 기반 흡연 여부 예측

건강검진 기반 구조화 데이터를 활용해 흡연 여부를 예측하고, 동시에 흡연이 생체지표에 남기는 신호를 어떻게 해석할 수 있는지 탐색한 머신러닝 프로젝트입니다. 이 프로젝트의 핵심은 단순히 점수를 높이는 데 그치지 않고, "잘 맞히는 문제"와 "의미 있게 읽는 문제"를 분리해 설계했다는 점에 있습니다.

## 프로젝트 개요

- 기간: 1주
- 형태: 4인 팀 프로젝트
- 문제 유형: 이진 분류
- 주제: 건강검진 생체지표만으로 흡연 여부를 어느 정도 추론할 수 있는가
- 핵심 결과: `RandomForest` 기반 실험에서 최고 `ACC 0.775`를 확인


## 1. 문제 정의

> 자기보고형 흡연 여부는 한계가 있는데, 건강검진에서 얻을 수 있는 생체지표만으로 흡연 여부를 어느 정도 추론할 수 있을까?

이 질문이 중요한 이유는 다음과 같습니다.

- 흡연 여부는 자기보고 방식에서는 과소보고될 가능성이 있습니다.
- 혈액검사, 지질수치, 간효소, 헤모글로빈 같은 생체지표에는 흡연의 누적 영향이 남을 수 있습니다.
- 예측 자체도 의미 있지만, 어떤 집단에서 흡연 신호가 더 강하게 나타나는지를 이해하면 의료 데이터 해석 관점에서도 가치가 있습니다.

이 프로젝트에서는 처음부터 문제를 두 갈래로 나누었습니다.

- 예측 문제: 실제로 얼마나 잘 맞힐 수 있는가
- 해석 문제: 어떤 지표와 어떤 집단에서 흡연의 흔적이 더 잘 드러나는가

이렇게 예측과 해석을 분리한 설계는, 단순한 모델 튜닝보다 연구 질문을 구조화해서 바라보고자 하였습니다.

## 2. 데이터 이해

### 사용 데이터

- `train.csv`: 7,000건
- `test.csv`: 3,000건
- 학습 데이터 컬럼 수: 18개
- 타깃 변수: `label` (흡연 여부)

주요 입력 변수는 다음과 같습니다.

- 나이
- 키, 몸무게, BMI
- 시력, 충치
- 공복 혈당, 혈압
- 중성지방, 콜레스테롤, HDL, LDL
- 혈청 크레아티닌
- 헤모글로빈
- 요 단백
- 간 효소율

라벨 분포는 다음과 같습니다.

- 비흡연: 4,429
- 흡연: 2,571

즉, 완전히 균형 잡힌 데이터는 아니며, 약 `2:1` 수준의 클래스 불균형이 존재합니다.

### 데이터의 특성

- 의료 도메인의 구조화 표 형태 데이터라서, 이미지나 텍스트보다 설명 가능한 변수 해석이 상대적으로 용이합니다.
- 한편 변수 간 상호작용과 비선형성이 존재할 가능성이 커서, 단순 선형 모델만으로는 충분하지 않을 수 있습니다.
- 연령과 BMI는 흡연의 직접 효과를 가리는 강한 교란 요인일 수 있어 별도 분석이 중요합니다.

### 데이터의 한계

- 흡연 여부 라벨이 자기보고 기반이라면 라벨 품질 자체에 한계가 있습니다.
- 단면 데이터이기 때문에 흡연의 누적 효과나 시간에 따른 변화를 직접 추적하기 어렵습니다.
- 건강검진 수치만으로는 생활습관, 병력, 약물 복용, 성별 기반 차이 같은 중요한 맥락이 충분히 반영되지 않을 수 있습니다.


## 3. 접근 방법

### 왜 이런 방식으로 접근했는가

하나의 지표만 좇지 않고, 목적에 따라 평가 기준과 모델 선택 전략을 달리했습니다.

#### 1) 예측 성능 중심 실험

- 대회 채점 기준인 `ACC`에 맞춰 모델을 최적화했습니다.
- 분류 threshold를 조정해 정확도를 높이는 전략을 사용했습니다.
- "실제로 점수가 얼마나 나오는가"를 우선적으로 보는 실험입니다.

#### 2) 해석 중심 실험

- 클래스 불균형의 영향을 줄이기 위해 `AUC`를 함께 사용했습니다.
- 단순 정확도보다, 흡연 신호가 생체지표에 얼마나 구분 가능하게 남는지를 더 균형 있게 보려는 목적이었습니다.
- "임상적으로 어떤 의미를 읽을 수 있는가"를 더 중요하게 본 실험입니다.

### 사용한 모델

노트북 기준으로 다음 모델들을 비교했습니다.

- Logistic Regression
- SVM
- RandomForest
- ExtraTrees
- AdaBoost
- XGBoost
- LightGBM
- CatBoost
- Voting Ensemble
- Stacking Ensemble

또한 다음과 같은 전처리 및 파생변수 아이디어를 함께 탐색했습니다.

- binning
- encoding
- clipping
- residualization(잔차화) 기반 파생변수
- 연령대 / BMI 기준 subgroup analysis

### 왜 RandomForest가 중심이 되었는가

최종적으로는 `RandomForest`가 가장 설득력 있는 기준 모델이었습니다.

- 구조화된 표 데이터에서 강한 베이스라인이 됩니다.
- 변수 스케일에 크게 민감하지 않아 다루기 쉽습니다.
- 비선형 관계와 변수 간 상호작용을 비교적 잘 포착합니다.
- 선형 모델보다 복잡한 패턴을 반영할 수 있고, 지나치게 공격적인 부스팅 모델보다 안정적인 비교 기준으로 쓰기 좋습니다.
- feature importance 기반의 해석 연결도 상대적으로 수월합니다.


## 4. 실험과 검증

### 실험 방식

- `GridSearchCV`
- `StratifiedKFold`
- `train/test split`
- threshold optimization

즉, 단일 모델 하나만 돌린 것이 아니라, 여러 모델을 같은 문제 설정 아래에서 비교하고 하이퍼파라미터 튜닝까지 수행했습니다.

### 성능 측정 방법

- 예측 성능 중심 실험에서는 `ACC`를 우선 기준으로 사용
- 보조 지표로 `AUC`, `F1`, `Precision`, `Recall`을 확인
- 해석 중심 실험에서는 클래스 불균형을 덜 왜곡하는 방향에서 `AUC`를 함께 확인

의료 데이터에서는 "높은 점수"가 곧 "좋은 모델"을 뜻하지 않기 때문에, 어떤 지표를 기준으로 모델을 선택할지, 설계관점에서 고민하고자 했습니다.

### 다른 방법과의 비교

여러 모델을 비교한 결과, 핵심 결과는 다음과 같습니다.

- 최고 성능 모델: `RandomForest`
- 최고 정확도: `ACC 0.775`

또한 평균 성능만 보지 않고, 집단별로 흡연 신호가 달라지는지 확인했습니다.

- 고령층에서는 `ACC 0.82` 수준으로 더 높은 예측력 확인
- 20-30대 저연령층에서는 `ACC 0.51` 수준으로 거의 무작위에 가까운 성능
- 저 BMI 집단에서는 `ACC 0.79`
- 고 BMI 집단에서는 `ACC 0.61`

이 결과는 흡연의 생체 신호가 모든 사람에게 동일하게 나타나는 것이 아니라, 연령과 BMI에 따라 더 강하거나 약해질 수 있다는 해석으로 이어졌습니다.


## 5. 회고

같은 데이터를 두고도 "점수를 높이는 문제"와 "의미를 해석하는 문제"를 구분해서 설계했다는 점이 만족스러웠습니다. 단순한 대회형 분류를 넘어서, 어떤 집단에서 신호가 더 잘 드러나는지까지 질문을 확장할 수 있었습니다.

반면 한계도 분명했습니다.

- 자기보고 기반 라벨로 인해 예측 성능의 상한이 뚜렷했습니다.
- 단면 구조화 데이터만으로는 흡연의 누적 효과를 직접 확인하기 어려웠습니다.
- 첫 프로젝트였던 만큼, 실험 결과 저장과 버전 관리가 체계적이지 못했습니다.
- 폴더 구조와 파일 명명 규칙이 일관되지 않아, 나중에 실험을 다시 추적하거나 비교하기가 어려웠습니다.

### 다시 한다면 이렇게 개선하고 싶습니다

- `cotinine` 같은 객관적 흡연 지표를 활용해 라벨 품질 개선
- 외부 데이터셋 검증으로 일반화 가능성 확인
- 종단 데이터 또는 반복 측정 데이터로 누적 효과 분석
- 실험별 설정, 점수, 해석 메모를 체계적으로 남기는 실험 로그 체계 도입
- 폴더 구조를 `data / notebooks / src / reports / outputs / experiments` 형태로 정리
- 필요하다면 `MLflow`, `Weights & Biases`, `DVC` 같은 도구로 실험 버전 관리 체계화

특히 처음 진행한 프로젝트라, 결과를 잘 저장하지 못했고 파일도 구조적으로 정리하지 못한 아쉬움이 남습니다. 그래서 이후에는 "모델 성능 향상"만큼이나 "실험을 재현 가능하게 기록하고 버전별로 비교 가능한 구조를 만드는 일"이 중요하다는 점을 분명히 배우게 되었습니다.
