## 프로젝트 구조
본 저장소는 생체지표를 활용한 흡연 여부 예측 프로젝트의 핵심 분석 과정과 주요 실험 결과를 담고 있습니다.

프로젝트는 탐색적 연구(exploratory research)의 성격을 가지며, 모델 설계와 평가 기준이 실험 과정에서 점진적으로 발전하였습니다.

현재 저장소에는 다음 두 개의 노트북이 포함되어 있습니다.
- age-bmi-group-train.ipynb
  
  나이대 및 BMI 그룹별로 데이터를 분리하여 모델을 학습, 평가함으로써,
  흡연 여부가 생체지표에 미치는 영향이 집단별로 어떻게 다르게 나타나는지 분석한 실험 노트북입니다.
- acc-threshold.ipynb
  
  accuracy 점수에 최적화되도록 threshold를 조정하고, serch를 통해 best model을 탐색한 노트북 입니다.
이 외에는 공통적으로 데이터 전처리, 기본 모델링, 성능 평가 등 프로젝트의 주요 분석 흐름을 담았습니다.



## 코드 구성
본 프로젝트는 고정된 파이프라인을 처음부터 설계한 방식이 아니라,

실험과 분석을 반복하며 문제 정의와 접근 방식을 정교화해 나간 연구 중심의 작업으로 진행되었습니다.

이 과정에서 일부 초기 실험 코드 및 중간 단계의 시도들은 하나의 통합된 스크립트 형태로 보존되지 않았습니다.

대신, 본 저장소에는 다음과 같은 목적을 중심으로 코드를 구성하였습니다.
- 프로젝트의 결론 도출에 직접적으로 기여한 최종 분석 흐름 보존
- 모델 성능 비교 및 해석 과정에서 중요한 의사결정 지점 명확화
- 포트폴리오에 기술된 결과와 인사이트가 재현 가능하도록 핵심 실험 유지

## 프로젝트의 맥락과 의의
이 저장소는 완성된 제품 수준의 코드베이스라기보다는,

문제 정의 -> 가설 설정 -> 실험 -> 해석이라는 연구과정을 충실히 반영한 연구 포트폴리오를 목적으로 합니다.

특히 본 프로젝트는 단순히 예측 정확도를 높이는 데에 그치지 않고,
- 예측 중심 모델과 해석 중심 모델 간의 차이
- 인구학적 요인(나이, BMI)이 생체지표 기반 예측에 미치는 영향
- 평가 지표 선택이 연구 목적에 따라 어떻게 달라져야 하는지
등을 함께 고민하며 진행하였고, 이러한 문제의식은 코드 구조와 분석 방식에도 반영되어 있습니다.
